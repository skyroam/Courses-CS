# Courses-CS
公开课翻译项目 
线上查看：[博客](https://www.wandersky.org/)的课程菜单。

本项目主要翻译一些计算机公开课，包括实验、作业、讨论和项目等内容，方便大家学习。对应的教材也会翻译，可以在[书栏](https://book.wandersky.org/)线上查看

当前进展：cs61a

比如cs61a的内容就分为以下目录：

- discussion 讨论
- lab 实验
- homework 作业
- project 项目
- textbook 教材
- solution 部分答案
- related 其他相关材料
